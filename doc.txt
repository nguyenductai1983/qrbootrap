1. Nhóm Core Laravel & PHP
Đây là bộ khung không thể thiếu để VS Code hiểu được cấu trúc thư mục và cú pháp của Laravel.

Laravel Extension Pack: Một bộ tổng hợp bao gồm Laravel Snippets, Laravel Blade Snippets, và Laravel Extra Intellisense.

PHP Intelephense: Cung cấp khả năng tự động hoàn thành (autocomplete) cực mạnh, nhảy đến định nghĩa hàm (Go to definition) và kiểm tra lỗi cú pháp PHP.

Laravel Blade Spacer: Tự động thêm khoảng trắng thông minh khi bạn gõ {{ }}.

2. Nhóm Livewire & Alpine.js
Vì Livewire và Alpine.js có những cú pháp đặc thù (như @click, wire:model), các extension này sẽ giúp highlight cú pháp chính xác.

Laravel Livewire Screenshots: Hỗ trợ autocomplete cho các directive của Livewire và hỗ trợ di chuyển nhanh giữa file Class và file View.

Alpine.js IntelliSense: Cung cấp gợi ý cho các thuộc tính của Alpine như x-data, x-init, x-on, giúp bạn không cần nhớ thủ công các directive.

3. Nhóm Giao diện (CSS & Bootstrap)
Dù bạn dùng Bootstrap, các công cụ hỗ trợ CSS vẫn rất quan trọng để quản lý class.

IntelliSense for CSS class names in HTML: Tự động gợi ý tên các class Bootstrap khi bạn gõ trong thuộc tính class="".

Bootstrap 5 Quick Snippets: Giúp gọi nhanh các component phức tạp của Bootstrap (như modal, navbar, card) chỉ bằng vài ký tự viết tắt.

Auto Rename Tag: Khi bạn đổi thẻ <div> thành <span>, nó sẽ tự động đổi thẻ đóng tương ứng, rất tiện khi dàn trang HTML phức tạp.

# 1. Cài đặt kho lưu trữ EPEL (nếu chưa có) và Supervisor
sudo dnf install epel-release -y
sudo dnf install supervisor -y

# 2. Khởi động và cho phép Supervisor tự bật khi server khởi động lại
sudo systemctl enable supervisord --now
sudo nano /etc/supervisord.d/reverb.ini
[program:reverb]
process_name=%(program_name)s
# Thay /var/www/qrbootrap bằng đường dẫn THỰC TẾ tới thư mục Laravel của bạn
command=php /var/www/qrbootrap/artisan reverb:start

# Tự động chạy khi Supervisor bật
autostart=true

# Tự động khởi động lại nếu Reverb bị crash (lỗi)
autorestart=true

# Thay bằng user đang quản lý thư mục web của bạn (vd: nginx, apache, root, hoặc tài khoản cpanel...)
user=root

# Lưu log (nhật ký) để dễ tìm lỗi
redirect_stderr=true
stdout_logfile=/var/www/qrbootrap/storage/logs/reverb.log
# Đọc file cấu hình mới
sudo supervisorctl reread

# Cập nhật hệ thống
sudo supervisorctl update

# Bật Reverb
sudo supervisorctl start reverb
sudo supervisorctl status
